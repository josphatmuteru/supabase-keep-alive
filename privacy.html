Privacy Policy for Supabase Keep Alive Extension

Last updated: August 27, 2025

Data Collection

This extension does not collect, store, or transmit any personal data to external servers. All configuration data (Supabase project URLs and ping settings) is stored locally in your browser only.

Data Usage

The extension uses your configured Supabase project URLs solely to send keep-alive requests to your own Supabase projects. No analytics, tracking, or data collection occurs.

Data Storage

All settings are stored locally using Chrome's storage API. This data remains on your device and can be deleted by uninstalling the extension.

Network Activity

The extension only sends HTTP requests to Supabase URLs you explicitly configure. No requests are made to third-party services.

Changes to Policy

Any changes to this privacy policy will be reflected in extension updates and posted at this URL.
