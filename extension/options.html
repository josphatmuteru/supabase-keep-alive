Supabase Keep Alive Settings

Extension Status

When enabled, the extension will automatically ping your Supabase URLs at the specified interval.

Ping Interval

Supabase free projects pause after 7 days of inactivity. Pinging every 24 hours is usually sufficient.

Supabase Project URLs

Add your Supabase project REST API URLs. Example: https://your-project.supabase.co/rest/v1/users
You can add a custom name for each URL and an optional anon key if authentication is required.
Each URL can have its own Supabase anon key. Find your anon key in: Supabase Dashboard → Settings → API → Project API keys → anon/public.
Tip: Use specific endpoints like /users or /posts to ensure real API activity.

Privacy & Security

Your privacy is protected:
• No data collection, tracking, or analytics
• All settings stored locally on your device only
• Supabase anon keys encrypted in Chrome's secure storage
• Network requests sent only to your configured URLs
• Open source - verify the code yourself
